🚀 快速开始

# 1. 克隆仓库
git clone https://github.com/zero00004/auto-resume-builder.git

# 2. 安装依赖
npm install

# 3. 生成示例简历
node index.js --demo

# 4. 用你的数据生成
node index.js --file resume.json

📝 输入格式

创建 resume.json 文件：

{
  "name": "张三",
  "title": "高级软件工程师",
  "contact": {
    "phone": "+86 138-0000-0000",
    "email": "zhangsan@email.com",
    "location": "北京"
  },
  "summary": "8年全栈开发经验...",
  "experience": [
    {
      "company": "ABC 科技",
      "position": "高级开发",
      "startDate": "2020-01",
      "endDate": "至今",
      "description": ["负责核心架构设计..."]
    }
  ],
  "skills": ["JavaScript", "Python", "React"],
  "education": [
    { "institution": "北京大学", "degree": "本科", "field": "计算机" }
  ]
}

📦 安装到全局

npm install -g auto-resume-builder

# 然后随处可用：
resume-builder --demo
